메뉴 -> VCS -> GitHub 프로젝트 공유
	- 저장소 이름 : 프로젝트명과 동일(임의) 	, * 공개(public) / 비공개(private)
	- 원격 : origin(기본값)
	- 설명 : README 표시될 메시지 (임의)

	[ 공유 ] 버튼 클릭

	2. 초기 커밋에 파일 체크 해서 [추가] 버튼 클릭
		- 특별한 경우가 아니면 모두 체크

	3. https://github.com/itdanjalog/tj_2024A_backend

3. ( 커밋 - 코드 수정할때마다 )
사이드 메뉴 -> 커밋 탭
	1. 커밋할 파일 체크 하기
	2. 메시지 작성하기
	3. [ 커밋 및 푸시 ] 클릭


4. gitHub 저장소에 있는 프로젝트 가져오기
	[ 인털리제이 최초 화면 ]
	1. VCS 에서 받기
	2. 계정없음 -> gitHub 로그인
	3. 로그인 성공/상태이면 원하는 저장소 선택후 [복제]
